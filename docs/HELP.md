# unframed 玩家手册

## 什么是 unframed？

unframed 是一个 AI 驱动的叙事游戏。没有预设的游戏机制——AI 从零开始构建世界观、规则、角色与剧情。你所做的一切都会影响故事走向。

## 快速开始

```bash
# 启动游戏
hatch run unframed           # 启动菜单（推荐）
hatch run unframed --tui     # 图形界面模式
hatch run unframed --seed seeds/xxx.md  # 跳过菜单直接加载种子
hatch run unframed --continue           # 继续上次游戏
```

## 游戏界面

### TUI 模式（`--tui`）

```
┌─────────────────────────────────────────────────┐
│  U N F R A M E D                    14:30:21    │
├────────────────────────────────┬────────────────┤
│                                │ 角色状态        │
│  叙事区域                       │  生命值: 100   │
│  （AI 的故事输出）               │  位置: 断指酒吧 │
│                                │                │
│                                │  回合 12       │
├────────────────────────────────┴────────────────┤
│  AI 正在构思剧情...                              │
├─────────────────────────────────────────────────┤
│  > 输入你的行动...                      [发送]   │
└─────────────────────────────────────────────────┘
```

- **左侧**：叙事输出区，显示 AI 生成的故事
- **右侧**：角色状态面板，显示 AI 标记为玩家可见的关键变量
- **底部**：输入框，输入你的行动或对话
- **状态栏**：AI 处理时显示当前操作状态

### TUI 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Ctrl+S` | 保存游戏 |
| `Ctrl+L` | 读取存档 |
| `Ctrl+D` | 删除存档 |

### CLI 模式（默认）

启动菜单让你选择：
- **新游戏**：选择一个种子（剧本）开始
- **加载存档**：从已保存的槽位继续
- **设置**：配置 API Key、模型等
- **文档**：查看此帮助

游戏中输入框支持以下命令：

| 命令 | 功能 |
|------|------|
| `/save` | 打开存档菜单 |
| `/save 1` | 快速存到槽位 1 |
| `/save list` | 列出所有槽位 |
| `/save <路径>` | 存到指定文件 |
| `/load` | 打开读档菜单 |
| `/load 1` | 快速读槽位 1 |
| `/delete` | 打开删档菜单 |
| `/quit` | 退出游戏 |

## 种子（Seed）

种子定义了游戏的世界观、时间、地点、角色、规则和目标。种子文件放在 `seeds/` 目录下，启动时自动识别。你也可以自己编写种子——参考 `docs/SEED_SPEC.md`。

## 游戏机制

### AI 是唯一的创世者

没有预设的规则、属性或战斗系统。AI 通过工具调用动态管理游戏状态。

### 状态即真相

游戏世界的所有状态都由 AI 通过工具调用读写。AI 的叙事必须与状态一致。

### 变量分层

AI 看到的变量分为三层：
- **核心区**：常驻变量，始终可见
- **活跃区**：最近操作过的变量
- **目录区**：其他变量，仅显示名称

### 右侧面板

AI 使用 `show_var` 工具主动将关键信息展示在右侧面板。只有 AI 认为你应该知道的信息才会显示。

### 剧情规划

AI 会提前规划剧情节点，并在推进故事时更新。被跳过的分支会被自动删除。

## 给玩家的建议

- **主动行动**：直接描述你的行动，AI 会根据你的选择推进剧情
- **探索世界**：可以随时询问环境信息（"这附近有什么？"）
- **没有预设规则**：你觉得合理的事情都可以尝试
- **改变方向**：想换个玩法直接说（"其实我是..."）
- **保存经常**：用 `Ctrl+S` 或 `/save` 定期保存，防止意外丢失

## 配置

通过 `--api-key`、`--base-url`、`--model` 参数或环境变量设置：

```bash
export OPENAI_API_KEY="sk-..."
export OPENAI_BASE_URL="https://api.example.com/v1"
export OPENAI_MODEL="gpt-4o"
```
