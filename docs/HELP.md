# unframed 玩家手册

## 什么是 unframed？

unframed 是一个 AI 驱动的叙事游戏。没有预设的游戏机制——AI 从零开始构建世界观、规则、角色与剧情。你所做的一切都会影响故事走向。

## 快速开始

```bash
# 启动 TUI（默认）
unframed

# CLI 模式
unframed --cli

# 跳过菜单直接加载种子
unframed --seed seeds/xxx.md

# 继续上次游戏（CLI 模式）
unframed --cli --continue
```

- **左侧**：叙事输出区，AI 生成的故事以 Markdown 渲染

- **状态栏**：AI 处理时显示当前操作（如"正在修改世界状态"）

### CLI 模式

启动菜单：

| 选项 | 说明 |
|------|------|
| **新建存档** | 输入存档名 → 选择种子 → 开始新游戏 |

| 命令 | 说明 |
|------|------|
| `/save` | 打开存档菜单（输入存档名） |

## 种子（Seed）

种子是游戏的"剧本文件"，定义了世界观、时间、地点、角色、规则和目标。每个种子由一对文件组成：`seeds/<名称>.json`（标题索引）和 `seeds/<名称>.md`（剧本内容）。

内置种子：

```
赛博朋克：深渊回响  — 2087 年，永夜之城"新重庆"
潮汐监狱           — 2147 年，深海环形海上堡垒
锈蚀边境：绿洲协议  — 2147 年，核冬天后的废土求生
孤星：坠落日       — 2247 年，殖民船坠毁后的外星求生
霓虹裁决           — 2140 年，同步轨道升降梯内
```

你也可以自己编写种子——参考 `docs/SEED_SPEC.md`。

## 游戏机制

### AI 是唯一的创世者

没有预设的规则、属性或战斗系统。AI 每轮先构思剧情方向，更新所有游戏状态变量，检查一致性，然后输出叙事。

### 状态即真相

游戏世界的所有状态都由 AI 通过工具调用读写。AI 的叙事必须与状态一致。右侧面板显示的是 AI 标记为"对玩家可见"的变量。

### 变量分层

AI 看到的变量分为三层：
- **核心区**：常驻变量，始终可见（如玩家目标）
- **活跃区**：最近操作过的变量（最多 20 个）
- **目录区**：其他变量，仅显示名称

### 剧情规划

AI 会构建一棵剧情树，提前规划可能的走向。被跳过的分支自动修剪。当故事到达结局节点时结束。

## 存档

- 存档文件存储在 `~/.unframed_saves/<uuid>.json`
- 每轮自动保存到当前存档（TUI 和 CLI 均支持）
- 主菜单"继续上一个存档"可直接回到上次游戏
- 存档文件存 JSON，包含：回合数、变量、对话历史、剧情树

## 配置

配置文件 `~/.unframed_config.json` 自动加载（如不存在则用默认值）。

读取优先级：**CLI 参数 > 环境变量 > 配置文件 > 内置默认**

```bash
export OPENAI_API_KEY="sk-..."
export OPENAI_BASE_URL="https://api.example.com/v1"
export OPENAI_MODEL="gpt-4o"
export OPENAI_TEMPERATURE="0.7"
```

## 给玩家的建议

- **主动行动**：直接描述你的行动，AI 会根据你的选择推进剧情
- **探索世界**：可以随时询问环境信息（"这附近有什么？"）
- **没有预设规则**：你觉得合理的事情都可以尝试
- **改变方向**：想换个玩法直接说（"其实我是..."）
- **保存经常**：用 `Ctrl+S` 或 `/save` 定期保存
